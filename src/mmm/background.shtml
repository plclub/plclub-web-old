Isabelle

Roadmap: Make a good pass of the complete tutorial doing the examples yourself before trying anything more complex: For most things that one wants to do when programming in Isabelle/HOL, there is the appropriate way to do it somewhere in the Tutorial. Having seen the complete tutorial is very important. The tutorial covers the traditional apply script syntax.

Once one becomes familiar with the way tactics and apply scripts work, he can read the second document about Isar and re-structure his proofs in the much more powerful Isar style.

The Isabelle/HOL distribution comes with a very nice set of examples, that include programming languages proofs.

  2. T. Nipkow. "Structured Proofs in Isar/HOL." [pdf]
  3. Isabelle Documentation
